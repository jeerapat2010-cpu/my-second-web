กลุ่ม 2 = ง่ายที่สุด! ตัด る แล้วเติม て เลย
กลุ่ม 2 ง่ายมากค่ะ — ไม่มีการเปลี่ยนเสียง ไม่มีข้อยกเว้น:
たべる → ตัด る → たべ → เติม て → たべて ✓
みる → ตัด る → み → เติม て → みて ✓
おきる → ตัด る → おき → เติม て → おきて ✓

กลุ่ม 3 จำ 2 ตัวนี้ให้ขึ้นใจ:
する → して | くる → きて

เคล็ดลับ: する ผัน si → shi + te | くる ผัน ku → ki + te (เสียงเปลี่ยนทั้งรากศัพท์!)

た形 — 過去形の作り方と使い方

ข่าวดีที่สุด: รูปอดีต (た形) สร้างง่ายมาก เพียงเปลี่ยน て → た และ で → だ ไม่ต้องเรียนกฎใหม่เลยค่ะ!

กฎรูปอดีต — ง่ายที่สุดในภาษาญี่ปุ่น!
รูปอดีต = รูปที่ เปลี่ยน て → た / で → だ

のんで → のんだ (ดื่มแล้ว)
たべて → たべた (กินแล้ว)
いって → いった (ไปแล้ว)
して → した (ทำแล้ว)
きて → きた (มาแล้ว)

จำได้ง่ายเลย — เรียน て เรียบร้อยแล้ว ก็ได้ た มาฟรี!

⚠️ ระวัง: た形 ไม่เหมือน ました (ทางการ) — た形 ใช้ในภาษาพูดธรรมดา

⏱️
〜ている vs 〜た — กำลัง vs เสร็จแล้ว
ている vs た — 動作の進行・完了・状態を表す

ทั้งสองรูปมาจาก รูปที ทั้งคู่ แต่ให้ความหมาย ต่างกันมาก 〜ている = กำลัง/อยู่ในสภาพนั้น | 〜た = เสร็จแล้ว/เกิดขึ้นแล้วในอดีต

🕐 Timeline ของรูปต่างๆ
ている มีความหมาย 3 แบบ — อย่าสับสน!
① กำลัง... (Progressive) กับกริยา action:
 いま ご飯を たべている = ตอนนี้กำลังกินข้าวอยู่

② มีสภาพ... (Resultant State) กับกริยา change-of-state:
 けっこん している = แต่งงานอยู่ (ไม่ใช่ "กำลังแต่งงาน"!)
 でんき が ついている = ไฟเปิดอยู่ (เปิดแล้วและยังเปิดอยู่)

③ อาชีพ/กิจวัตร (Habitual):
 かいしゃで はたらいている = ทำงานอยู่ที่บริษัท (อาชีพ)

คนไทยมักสับสน ① กับ ② — กริยา change-of-state = สภาพปัจจุบัน ไม่ใช่ progress!
