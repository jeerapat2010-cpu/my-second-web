ฮิรางานะ 46 ตัวพื้นฐาน
ひらがな清音 — 基本46文字

ฮิรางานะเป็นตัวอักษรโค้งมนที่มาจากการย่อตัวคันจิโบราณ แต่ละแถวเริ่มด้วยพยัญชนะแล้วตามด้วยเสียงสระ あいうえお (a-i-u-e-o) คลิกที่ตัวอักษรใดก็ได้เพื่อฟังเสียงค่ะ

เทคนิค: 5 สระหลัก あいうえお
สระญี่ปุ่นตรงกับไทยเกือบทั้งหมด — จำแค่ 5 เสียงก็ใช้ได้แล้ว!

あ (a) = อา — ปากเปิดกว้าง เหมือนหมอดูฟัน
い (i) = อิ — ยิ้มกว้างๆ เหมือนพูดว่า "อิ"
う (u) = อุ — ริมฝีปากไม่ยื่น แค่พยักหน้าเงียบๆ (≠ อู ในภาษาอังกฤษ)
え (e) = เอะ — คล้าย "เ" ในไทย ปากกึ่งเปิด
お (o) = โอะ — ปากกลม แต่ไม่ยื่นเหมือน "โอ" อังกฤษ

💡 จำลำดับ: "อา-อิ-อุ-เอ-โอ" ร้องซ้ำๆ จนติดปาก!
แถว あ～な · เสียงพื้นฐาน 25 ตัว
—a—i—u—e—o
แถว は～ん · เสียงพื้นฐาน 21 ตัว
—a—i—u—e—o
⚠️ ตัวอักษรที่ต้องระวังเป็นพิเศษ:
• は (ha) — ปกติอ่าน "ฮะ" แต่เมื่อเป็นอนุภาค อ่านว่า "วะ"
• へ (he) — ปกติอ่าน "เฮะ" แต่เมื่อเป็นอนุภาค อ่านว่า "เอะ"
• を (wo) — ใช้เฉพาะเป็นอนุภาค อ่านเพียง "โอะ"
• ん (n) — เสียงเปลี่ยนตามตัวถัดไป (ดูรายละเอียดในส่วนเสียงพิเศษ)

覚え方 — 形のコツ

วิธีที่ดีที่สุดคือ "ผูกเรื่อง" เชื่อมรูปร่างตัวอักษรกับเสียง นักวิจัยพบว่าวิธีนี้ช่วยให้จำได้ 3 เท่า เร็วกว่าการท่องซ้ำๆ ค่ะ

เทคนิคแถว か (ka-row) — เสียง ค
か (ka) — มองดูคล้ายคนกำลัง "ค้าง" ขา ท่าเตะกังฟู → "คะ"
き (ki) — มี 4 ขีดคล้าย "กีตาร์" มี 4 สาย → "คิ"
く (ku) — คล้ายจงอยปากนก "คุ้ง" น้ำ → "คุ"
け (ke) — คล้าย "เค" โบราณ มีขาข้างหนึ่ง → "เคะ"
こ (ko) — สองขีดแนวนอน คล้ายรอยเท้า "โค" (วัว) → "โคะ"
เทคนิคแถว さ (sa-row) — เสียง ซ/ช
さ (sa) — คล้ายตัว ซ ที่มีขาข้างเดียว → "ซะ"
し (shi) — คล้ายเบ็ดตกปลา 🪝 → เบ็ด"ชิ้น" → "ชิ"
す (su) — คล้ายอ่างน้ำ "ซู" น้ำ → "ซุ"
せ (se) — คล้ายตัว ฅ ไทย มีหัว มีขา → "เซะ"
そ (so) — คล้ายตัว Z ที่บิดตัว คล้ายงูโซ่ → "โซะ"
เทคนิคแถว た (ta-row) — เสียง ท/ช/สึ
た (ta) — คล้ายรากไม้ "ทะ" สาขาที่แตกออก → "ทะ"
ち (chi) — คล้ายตัวเลข 5 มีงวง เหมือนงู"ชิ"ว → "ชิ"
つ (tsu) — คล้ายตัว つ เล็กๆ บิดงอ เสียง "สึ" = ฟันหัก "tsss" → "สึ"
て (te) — คล้ายตะขอแขวน "เท" เสื้อผ้า → "เทะ"
と (to) — คล้ายตรีศูล หรือ "โต้" พลั่ว → "โทะ"
เทคนิคแถว な/は/ま/ら/わ
な (na) — เหมือน "น้า" ที่กำลังโบกมือ มีส่วนโค้ง → "นะ"
に (ni) — เหมือนคนยืนมือ 2 ข้าง คล้ายตัว ⊢ → "นิ"
は (ha) — คล้าย "ฮ" มีขาสามขา → "ฮะ"
ふ (fu) — คล้ายหน้าคนกำลัง "ฝ่า" ลม เป่าลมเบาๆ → "ฟุ"
ら (ra) — คล้ายตะขอโค้งขนาดใหญ่ ลิ้น"ระ"วัง → "ระ"
り (ri) — เหมือนตัว り สองขีดเอียง ลิ้น"ริ"น → "ริ"

ตัวขุ่น 濁音 + กึ่งขุ่น 半濁音
濁音・半濁音 — 声を変える記号

เพียงเพิ่ม ゛ (nigori/dakuten) ที่มุมบนขวาของตัวอักษร จะทำให้เสียงเปลี่ยนจาก "ไม่ก้อง" เป็น "ก้อง" ทำให้เกิดเสียงใหม่อีก 20 เสียงโดยไม่ต้องจำรูปร่างใหม่!

หลักการตัวขุ่น — ง่ายมาก!
ลองจับคู่เสียงไทยกับญี่ปุ่น:

か行 (k) + ゛ → が行 (g): คะ → กะ | คิ → กิ
さ行 (s) + ゛ → ざ行 (z): ซะ → ซะก้อง | ชิ → จิ
た行 (t) + ゛ → だ行 (d): ทะ → ดะ | เทะ → เดะ
は行 (h) + ゛ → ば行 (b): ฮะ → บะ | ฮิ → บิ
は行 (h) + ゜ → ぱ行 (p): ฮะ → ปะ | ฮิ → ปิ

💡 ゜ (handakuten = วงกลมเล็ก) ใช้กับแถว は เท่านั้น ให้เสียง ป

ตัวผสม 拗音 — เสียงสองตัวรวมกัน
拗音 — きゃ・しゃ・ちゃ ...

ตัวผสมเกิดจากนำ や・ゆ・よ ขนาดเล็ก มาต่อท้ายตัวอักษรแถว "i" เช่น き + ゃ (เล็ก) = きゃ (kya) ออกเสียงพร้อมกันในพยางค์เดียว

ตัวเล็ก vs ตัวใหญ่ — ต่างกันมาก!
きや (ki-ya) = 2 พยางค์ → คิ + ยะ = "คิ-ยะ"
きゃ (kya) = 1 พยางค์ → คยะ = "คยา" (เร็วๆ รวมกัน)

จำง่ายๆ: ตัวเล็ก = รวมเป็นพยางค์เดียว, ตัวใหญ่ = แยกสองพยางค์

ตัวผสมที่ใช้บ่อยที่สุด: しゃ(sha) しゅ(shu) しょ(sho) — เช่น しゃしん(shashin)=ภาพถ่าย

カタカナ完全版 — 外来語の文字

คาตากานะมีเสียงเหมือนฮิรางานะทุกตัว แต่รูปร่างต่างออกไป (เส้นตรงกว่า เหลี่ยมกว่า) ใช้เขียนคำทับศัพท์จากภาษาต่างชาติ ชื่อต่างชาติ และเสียงประกอบ (เช่น เสียงสัตว์)

วิธีจำคาตากานะจากฮิรางานะที่รู้แล้ว
หลายตัวมาจากการ "ตัดทอน" ฮิรางานะ:

あ → ア: เอาส่วนขวาบนมา | い → イ: เอาแค่เส้นขวา
う → ウ: ตัดส่วนบน เหลือแค่โค้ง | か → カ: ตัด curve ออก
き → キ: เส้นตรงทั้งหมด | し → シ: พลิกด้านข้าง
す → ス: เอาส่วนบน | て → テ: ทำให้ตรงขึ้น

⚠️ คู่ที่สับสนบ่อยที่สุด:
シ (shi) vs ツ (tsu): จุดอยู่ซ้าย-ขวาต่างกัน
ソ (so) vs ン (n): ทิศทางเส้นทะแยงต่างกัน
ワ (wa) vs ウ (u): ワ มีขาลงสองข้าง
คาตากานะ ア～ン ครบ 46 ตัว

コーヒー (koohii) = กาแฟ | スマートフォン (sumaatofon) = สมาร์ทโฟน | ケーキ (keeki) = เค้ก

ฮิรางานะ vs คาตากานะ — ใช้เมื่อไหร่?
ひらがなとカタカナの使い分け
ใช้ ฮิรางานะ ใช้ คาตากานะ คำญี่ปุ่นแท้ เช่น さくら (sakura) คำทับศัพท์ต่างชาติ เช่น コーヒー (koohii = กาแฟ) ไวยากรณ์และอนุภาค เช่น は・を・に ชื่อต่างชาติ เช่น タイ (Tai = ไทย) คำอ่านของคันจิ (furigana) เช่น 日本（にほん） เสียงสัตว์/ประกอบ เช่น ワンワン (wan-wan = โฮ่ง) หนังสือเด็ก ข้อความที่ไม่ใช้คันจิ เน้นคำ/สไตล์ (เหมือน bold หรือ italic) คำกริยา คุณศัพท์ ส่วนที่ผันรูป ชื่อบริษัท แบรนด์สินค้าญี่ปุ่น
🌸 ตัวอย่างประโยคที่ใช้ทั้งสามระบบพร้อมกัน:
私はコーヒーが好きです。
Watashi wa koohii ga suki desu. = ฉันชอบกาแฟค่ะ

• 私 = คันจิ (ตัวเลขสำคัญ)
• は・が・です = ฮิรางานะ (ไวยากรณ์)
• コーヒー = คาตากานะ (คำทับศัพท์จากอังกฤษ)
